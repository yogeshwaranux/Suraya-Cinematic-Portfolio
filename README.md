# Suraya Video Portfolio

A responsive React portfolio website built with Vite, Tailwind CSS, and React. It showcases video showreels, YouTube Shorts, Instagram highlights, client testimonials, and direct contact options for email and WhatsApp.

## Features

- React + Vite UI
- Tailwind CSS styling
- Embedded YouTube and YouTube Shorts videos
- Instagram preview gallery
- Contact section with email and WhatsApp links
- Responsive layout for desktop and mobile

## Project Structure

- `src/` — React source files
- `src/App.jsx` — main portfolio page and sections
- `src/main.jsx` — app entry point
- `src/styles/globals.css` — global styling
- `public/` — static assets (if used)
- `dist/` — production build output

## How to Run

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`

## Build

```bash
npm run build
```
